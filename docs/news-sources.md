## Notes on Implementation

- **RSS Polling**: Most feeds should be polled every 15-30 minutes. High-volume feeds (Common Dreams, Al Jazeera) may warrant 5-10 minute intervals.
- **Deduplication**: Many progressive outlets cover the same stories. Implement title/URL deduplication.
- **Substack Feeds**: Append `/feed` to any Substack publication URL for RSS.
- **Content Extraction**: Some feeds provide full content, others only summaries. May need article scraping for full text.
- **Reuters Workaround**: Since Reuters killed RSS, consider using their API or a third-party feed generator.
- **Feed Validation**: All feed URLs should be verified before production deployment — some may have changed since this document was compiled.
